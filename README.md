# NIT Bhopal – MBC Portal App

### A unified platform for the MBC Department to manage academics, streamline administration, and enhance communication.  
Easily organize classes, manage students and faculty, track attendance, assess performance, and share feedback.  
Access academic records, view marks, and collaborate seamlessly.

---

## About

The **MBC Portal App** is a full-stack web application developed using the **MERN (MongoDB, Express.js, React.js, Node.js)** stack.  
It is designed to meet the academic and administrative needs of the **MBC Department, NIT Bhopal**, enabling smooth class organization, academic evaluation, and communication between students, faculty, and administrators.

---

## Features

- **Role-Based Access**
  - **Admin:** Manage faculty, students, subjects, and departmental settings.
  - **Teacher:** Take attendance, assign marks, provide feedback, and communicate with students.
  - **Student:** View marks, attendance records, and performance reports.

- **Admin Dashboard**
  - Add/edit/remove students and faculty.
  - Create and manage classes and subjects.
  - Oversee departmental operations.

- **Attendance Tracking**
  - Teachers can mark attendance for each class.
  - Generate attendance summaries and reports.

- **Performance Assessment**
  - Enter marks and feedback for students.
  - Students can track progress over time.

- **Data Visualization**
  - Interactive charts and tables for performance insights.
  - Clear, student-friendly progress tracking.

- **Communication Tools**
  - In-app messaging between teachers and students.
  - Quick and organized information exchange.

---

## Technologies Used

- **Frontend:** React.js, Material UI, Redux  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB
